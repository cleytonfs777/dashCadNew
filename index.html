Como este painel ajuda a decisão

operacional • tempos • carga
O foco aqui é tempo-resposta real. A partir dos status do CSV (despachado, a_caminho, no_local, em_controle, terminado), o painel calcula:
  • Despacho → Saída: despachado → a_caminho
  • Saída → Chegada: a_caminho → no_local (indicador direto de deslocamento real)
  • Chegada → Controle: no_local → em_controle (tempo até estabilizar)
  • Controle → Encerramento: em_controle → terminado
  • Ligação → Inclusão: data_hora_ligacao → chamada_data_hora_inclusao (processo interno)
Use os filtros de natureza, unidade, recurso e viatura para identificar gargalos e saturação.
